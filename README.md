# Next Clean Boilerplate

Minimalist Next.js boilerplate with modern web development tools.

## Features

- Next.js 15
- TypeScript
- pnpm
- Tailwind CSS
- shadcn/ui
- Configured fonts
- ESLint
- Prettier
- VS Code configuration
- GitHub Actions CI

## Important scripts

- `pnpm check`: Runs lint, typecheck and formatting concurrently.
